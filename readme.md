# JidaCalendar

Api para manejo de calendario.


### Ejemplo instancia:

```
const calendar = new JidaCalendar({});
```

#### Parametros

**specs**: [Object] Objeto de parametros de configuracion, puede recibir
las siguientes propiedades
- year
- month
- day
- locale


#### propiedades

- ready
- locale
- year
- day
- view
- VIEWS
- month
- dayjs
- daysNames
- months

#### Metodos

- nextMonth
- nextYear
- nextWeek
- previousMonth
- previousYear
- previousWeek
- changeMonth
- changeYear
- changeWeek
- ChangeView
- setView
